Eines que utilitzarem

Per poder començar a treballar és important preparar les eines necessàries. Pel disseny d’interfícies web necessitem un editor de codi o IDE amb reconeixement de codi i complements adients per a poder treballar còmodament amb HTML i CSS. També serà interessant comptar amb eines per a poder planificar els esbossos i maquetes. Finalment, com a tot procés de desenvolupament serà important tenir un sistema de control de versions (VCS).

Visual Studio Code

Entorn de desenvolupament versàtil utilitzat per la comunitat per diferents llenguatges de programació i de marques.

VS code
Plugins

Instil·lació de plugins necessaris dins de VS Code. Faran que la nostra feina sigui més àgil i fàcil.

Més Info
  • Live Server, servidor local en temps real
  • HTML snippet, per l’escriptura ràpida de codi
  • HTML End Tag Labels, ens ajudarà a visualitzar bé l’estructura de l’HTML
Wireframes i Mockups

Abans de codificar és important comptar amb un Wireframe i un Mockup inicials. Figma ens pot ser de gran ajuda

Figma
Control de versions Git i GitHub

En qualsevol desenvolupament és important tenir control de versions així com col·laboració entre diferents persones. Git i un servei de repositoris en línia com GitHub ens ho permeten.
