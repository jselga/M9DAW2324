Per poder començar a treballar és important preparar les eines necessàries. Pel disseny d’interfícies web necessitem un editor de codi o IDE amb reconeixament de codi i complements addients per a poder treballar comodament amb HTML i CSS. També serà interessant comptar amb eines per a poder planificar els esboços i maquetes. Finalment, com a tot procès de desenvolupament serà important tenir un sitema de control de versions (VCS).
